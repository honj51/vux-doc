# 1px边框解决方案

## 应用场景

1px边框问题来源于在Retina屏幕下边框宽度大于1像素，略丑，解决方案来源于`FronzenUI`

一般用于`九宫格`等布局，在vux中配合`Flexbox`使用，常见于电商首页，支付工具服务列表页

## 使用

``` html
<style lang="less">
@import '~vuxx/src/styles/1px.less';
</style>
```
根据位置可用类名为：
+ `vux-1px-t`
+ `vux-1px-b`
+ `vux-1px-tb`
+ `vux-1px-l`
+ `vux-1px-r`
