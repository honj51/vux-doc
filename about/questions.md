## Vux和Vuex名字太像了

## 可以在template中使用其他标签吗

当然可以，局部注册组件的时候指定名字。

``` js
{
  components: {
    xalert: Alert
  }
}

```

如果使用全局注册，则

``` js
Vue.component('xalert', vuxAlert)
```

## 为什么部分组件要加x-前缀

若不更名，可能在开发时与标准html标签相同而导致冲突或者bug。
