# 开始使用vux之前

我们假设你应该已经写过`vue`页面或者看过文档，否则在使用过程中会遇到很多疑惑。这种疑惑的锅vux不背 :doge:。

如果以下概念有不清楚的地方，需要再次查看文档，聪明如你，肯定花不了多少时间：

+ `slot` 内容插槽
+ `prop`
+ `v-model`
+ `@click` `events` `$emit`
