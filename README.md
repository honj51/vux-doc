# Vux

Vux 基于`Vue`和`WeUI`的组件库。

Vux的目标是：

+ 尽量满足手机端页面大部分组件需求。
+ 尽量保持小体积(甚至不依赖`Zepto`)。
+ 尽量简洁优雅。

### 文档修订



``` bash
# clone the doc repo
git clone https://github.com/vuxjs/vux-doc.git

# 安装gitbook-cli
npm install gitbook-cli -g

# cd到文档目录
gitbook install
gitbook serve

```
